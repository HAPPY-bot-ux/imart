iPhoneMart - Example 7-page website
Files included:
- index.html (home)
- products.html (catalogue)
- product.html (product details template)
- about.html
- services.html
- blog.html
- contact.html
- css/style.css
- js/main.js
- assets/logo.svg, whatsapp.svg, bg.svg

How to use:
1. Download and unzip the folder.
2. Open index.html in your browser (no server required).
3. Replace assets/bg.svg with a real background image if desired.
4. IMPORTANT: Replace the WA_NUMBER in js/main.js with your WhatsApp number in international format (e.g. 2761xxxxxxx).
